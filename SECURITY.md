# Security Policy

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, email **safarishahim@gmail.com** with details and, if possible, a
reproduction. You'll get an acknowledgement within a few days, and a fix or
mitigation plan as soon as the issue is confirmed.

## Supported versions

The latest published `0.x` release receives fixes. Once `1.0` ships, the latest
minor will be supported.
